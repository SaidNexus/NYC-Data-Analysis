# 🗽 NYC Data Analysis Project

## 📌 Overview
This project involves performing **exploratory data analysis (EDA)** on a real-world dataset from New York City. The goal is to uncover insights, detect patterns, and support data-driven decision-making through clear visualizations and statistics.

---

## 🛠️ Tools & Technologies

- **Python**
- **Pandas** – Data wrangling & manipulation  
- **Matplotlib** & **Seaborn** – Visualization  
- **Jupyter Notebook**

---

## 🔍 Key Steps

1. Data cleaning (handling missing values, formatting)
2. Univariate & bivariate analysis
3. Identifying trends and customer behavior
4. Visual comparisons across branches and time periods
5. Insightful graphs and observations
